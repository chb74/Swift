### Xcode 
앱개발을 위한 코드 편집기 

* 스토리보드 : 앱디자인이 가능함. 화면 전환도 표현 
* 시뮬레이터 : xcode에서 작성한 코드를 띄워서 화면 확인 가능 (버그, 디자인 확인) 
* Machine Learning 기능을 제공해준다. 
* AR (증강현실)
